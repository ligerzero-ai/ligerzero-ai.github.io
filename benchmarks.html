Interatomic Potential Benchmarks

A collection of curated benchmarks and datasets for evaluating the accuracy of interatomic potentials and machine learning force fields (MLFFs). These resources are essential for validating physical quantities such as elastic constants, surface energies, and structural stability.

Accuracy Evaluation Resources

Matbench

An ImageNet for materials science. It provides benchmark tasks for predicting various material properties from crystal structure using machine learning.

Machine Learning Materials Project

JARVIS-DFT Benchmarks

NIST's Joint Automated Repository for Various Integrated Simulations (JARVIS) provides a large database of DFT calculations for benchmarking interatomic potentials.

NIST DFT Potential Validation

ColabFit

A collaborative project for fitting and benchmarking interatomic potentials, providing tools and datasets for consistent evaluation of ML potentials.

ML Potentials Fitting

FLARE Benchmarks

Benchmarks focused on on-the-fly active learning of interatomic potentials, useful for evaluating performance in dynamic simulation environments.

Active Learning Molecular Dynamics

NOMAD Repository

The NOMAD Lab provides a vast repository of computational materials science data, including benchmarks for various physical properties across many simulation codes.

Open Data FAIR Principles
