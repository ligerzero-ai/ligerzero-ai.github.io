This page provides an overview of the various datasets I have developed and maintained, ranging from grain boundary segregation studies to high-throughput unary structural databases.

The ASSYST Methodology

Automated Small SYmmetric Structure Training (ASSYST)

Based on the work by Poul, Huber & Neugebauer published in npj Computational Materials 11, 174 (2025), ASSYST is a strategy for generating unbiased and systematically extendable training data for machine learning interatomic potentials (MLIPs) for multicomponent alloys.

Key Features

  • Explores the full space of random crystal structures with space groups using PyXtal
  • Only small cells (~10 atoms) are needed for the DFT training set
  • Training set size and completeness can be systematically controlled with very few parameters
  • Generates training data in one shot without iterative active learning refinement
  • Validated on binary and ternary phases, random alloys, point and extended defects, and phase diagrams

Available Datasets

Ni-X GB Segregation + Cohesion

A comprehensive dataset exploring the segregation of various alloying elements (X) to Nickel grain boundaries and their subsequent impact on grain boundary cohesion.

Method: DFT (VASP)

Fe-X GB Segregation + Cohesion

High-throughput data for solute segregation in Ferritic Iron grain boundaries. Focuses on the competition between different solutes and their role in embrittlement.

Method: DFT (VASP) — KP method, PAW-PBE

Coverage: 91 solute elements, 6 grain boundaries, 28 substitutional sites

Records: 9,131 total (4,897 unrelaxed + 4,234 relaxed)

Ni-X ASSYST Dataset

Dataset generated using the ASSYST framework for Ni-based systems, capturing a wide range of structural motifs and configurations for potential development.

Method: ASSYST + DFT

Fe-X ASSYST Dataset

Exploration of the Fe-X configurational space using the ASSYST methodology, providing a rich set of structures for complex Fe-based alloys.

Method: ASSYST + DFT

Unary Periodic Table Dataset

A massive collection of structural data for unary systems across the entire periodic table, designed for benchmarking and general-purpose potential training.

Coverage: Elements 1-118
